Fundamentals of Web-Based Multimedia Communications, Designing for Bodies introduces responsive web design using HTML, CSS, and Javascript, design fundamentals, the project planning process, as well as contemporary best practices. This course is will use lectures and labs to explore the creation and design theories behind multimedia information systems.

Labs will focus on the fundamentals of designing for the web using integrated authoring techniques with a focus on accessibility and critical design enabled by the underlying technologies that make them possible. The class will integrate learning how to code with understanding how code came to be and where we can look to imagine its future development and adaptation as integral to almost all areas of culture.
