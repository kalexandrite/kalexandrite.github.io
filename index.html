About this Course:

This course is will use lectures and labs to explore the creation and design theories behind multimedia information systems.
Labs will focus on the fundamentals of designing for the web using integrated authoring techniques with a focus on accessibility and critical design enabled by the underlying technologies that make them possible. The class will integrate learning how to code with understanding how code came to be and where we can look to imagine its future development and adaptation as integral to almost all areas of culture.


Class Schedule

Laboratory version of Information Science + Studies 240. Multimedia information systems, including presentation media, hypermedia, graphics, animation, sound, video, and integrated authoring techniques; underlying technologies that make them possible. Practice in the design innovation, programming, and assessment of web-based digital multimedia information systems. Intended for students in non-technical disciplines.
